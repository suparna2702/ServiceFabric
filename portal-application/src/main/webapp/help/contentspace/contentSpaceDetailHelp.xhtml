Introduction

Top

The Content Workspace is your repository of documents that you choose to share with employees, Partner Program members. In the Content Workspace, you can:

Upload/download documents

When you upload a document with the same name, a new version is assigned.

Share Documents with those in the network

View the history of the document

Viewers (those in the document share list) can Download the document (the workspace keeps track of what version was downloaded)

Be Alerted to updated posting of a document ,e.g. new revisions

Comment/Annotate to provide input

View other comments/annotations

View the document history

Creating a Content Workspace

Top

This section describes how to navigate to a Content Workspace and create one:

Select Collaborate from the Main Menu

Select Content from the dropdown menu

Select the [+] icon to create a Content workspace and input the workspace name and description in the pop-up box.

Select View to see the created Content workspace.

View the created Content Workspace.

Upload Content

Top

This section describes how to upload a content in the Content Workspace:

Select the Up Arrow icon to upload a content

Select the colored ball icon to upload a content from your local drive

Your local drive file directory will pop-up. Choose the file from your local file folder and select Open from the pop-up page. The file name will appear in the Upload Files box.

Select the Upload button, upload progress bar appears

Select Close

Select Save. A progress bar will indicate the % uploaded to the Content workspace.

View Uploaded Document

Assign document category, by selecting the pencil icon

A pop-up dialog box appears. Fill in the desired category information . Press the [+] to add the category. Then, select update.

View Content

Top

This section describes how to view a content in the Content Workspace:

You may view a content and comment

To comment, select the comment icon. A 2nd level menu will appear with tools : Comment, Draw, Highlight, Text, Strikeout
